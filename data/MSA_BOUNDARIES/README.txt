Notes:

This data is encoded using the Shapefile format. Using Geotools to read it.
Once an Object of class SimpleFeature is extracted from the shapefile, The MSA's ID is stored in properties named CBSAFP and GEOID (Both have the same value)
The MSA's ID allows us to link the region to the MSA
